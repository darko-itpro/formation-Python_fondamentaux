# Introduction aux tests

Pour toute information, consultez [l'aide mémoire destiné aux tests](../memo/tests/index.md).

## Création du module de la fonction

Créer un fichier `exos/base/media_utils_v1.py` et copiez-y la fonction `is_viewed(episode:list)`.

## Création des tests

Créez l'arborescence `tests/exos/base/test_is_viewed_v1.py` à partir de la racine du projet. Créez 
les tests dans ce fichier.
