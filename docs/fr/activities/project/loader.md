# Chargement de données

Suite à l'explicaiton sur les générateurs, modifiez la fonction pour en faire un générateur qui 
fournit chaque épisode à chaque itération.

Cette source de données contient des épisodes de plusieurs séries. En conséquence, créez une 
collection de séries (objets `TvShow`) contenant les objets correspondants en choisissant les 
structures de données les plus adaptées.
