# Découverte du langage

Dans les exercices suivants, il vous sera demandé d’afficher une valeur. **N’affichez que cette
valeur**, n’essayez pas de mettre en forme, cela sera la responsabilité d’autres composants.

## Fichiers de travail
Pour les exercices de cet énoncé, vous allez travailler avec les fichiers suivants :

 * `exos/base/exo_01a.py` que vous devrez créer.
 * `exos/base/exo_01b.py` qui vous est fourni.

---

## premier exercice
Pour votre premier code, commencez par créer un fichier source `exo_01a.py` dans le répertoire exos.

Votre formation a une durée de **5 jours**, chaque jour ayant une durée de **7 heures**.

Dans votre script :

 * affectez ces valeurs à des variables.
 * Affichez les valeurs de ces variables. 
 * Calculez la durée totale en heure de votre formation et affichez cette durée.

---

## Second exercice
Vous pouvez répondre à cet exercice à la suite du code précédent dans le fichier source exo_01a.py.

La pause de midi du premier jour a durée **72 minutes**. Affichez combien cela fait en heure(s) et
minutes (vous pouvez évidemment afficher d’une part les heures et d’autre part les minutes)

---
