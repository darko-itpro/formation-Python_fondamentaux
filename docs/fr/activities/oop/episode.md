# Première amélioration

La classe épisode est *presque* bien réalisée. Assurez-vous que :

 * lors de la création d'un épisode, les attributs attendant des `int` doivent bien avoir des 
   données de type `int` ou `None` pour l'année et la durée.
 * Un épisode ne peut être créé si son numéro d'épisode ou de saison est négatif.
