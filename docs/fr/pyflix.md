# Le package Pyflix

Pour certains exercices, il vous est demandé d'utiliser le module `pyflix.datasource`. Ce module est
dans un package dédié qui est un projet à part, 
[pyschool-lib](https://github.com/darko-itpro/pyschool-lib).

Vous disposez d'une 
[documentation d'installation](https://darko-itpro.github.io/pyschool-lib/fr/installation/).

## Utilisation de la bibliothèque

Assurez-vous d'avoir installé la dernière version en suivant la 
[documentation d'installation](https://darko-itpro.github.io/pyschool-lib/fr/installation/).

Dans un script Python ou dans le shell intéractif (ou iPython), vous pouvez importer le module 
`pyflix.datasource` :

```python
import pyflix.datasource as ds
```

Vous disposez alors d'un certain nombre de fonctions qu'il vous sera demandé d'utiliser dans 
certains exercices.
