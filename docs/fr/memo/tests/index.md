# Les tests avec Pytest

Si les tests sont bien au programme d'une formation sur les *fondamentaux*, elle ne permet pas
d'entrer dans tous les détails.

Les tests sont un outil essentiel et fondamental pour n'importe quel développement.

## Installation de pytest

Pytest ne faisant pas parti de la distribution standard, il faut l'installer. Dans le cadre de ce
projet, pytest est dans le fichier `requirements.txt` du projet et a été installé à son
initialisation.

Pour un projet *neuf*, il faut l'installer avec :

```python
pip install pytest
```
