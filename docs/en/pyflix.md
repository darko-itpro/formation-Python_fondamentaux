# The Pyflix package

For some exercises, you are asked to use the `pyflix.datasource` module. This module is
part of a dedicated package that is a separate project, 
[pyschool-lib](https://github.com/darko-itpro/pyschool-lib).

You have access to 
[installation documentation](https://darko-itpro.github.io/pyschool-lib/en/installation/).

## Using the Library

Make sure you have installed the latest version by following the 
[installation documentation](https://darko-itpro.github.io/pyschool-lib/en/installation/).

In a Python script or in the interactive shell (or iPython), you can import the 
`pyflix.datasource` module:

```python
import pyflix.datasource as ds
```

You will then have access to a number of functions that you will be asked to use in 
certain exercises.
